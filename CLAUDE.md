# learning-forge

Turn experience into improvement. Capture learnings from sessions, route each one to the specific project, forge, or guardrail that needs it.

## Skills

| Skill | What it does |
|-------|-------------|
| `/learn` | Extract learnings from a session, classify, route to destinations, log |
| `/learn-review` | Aggregate learnings across ecosystem, find patterns, surface repeat offenders |

## Guardrails

- No software — skills and templates only
- Every learning must land somewhere specific (a file, not a conversation)
- Learnings must be actionable, not observations

## How It Connects

learning-forge is the bridge between experience and improvement:
- improve-forge finds what's wrong → learning-forge captures why and routes the fix
- brutal-forge demolishes code → learning-forge ensures the findings don't repeat
- Any session that produces insight → `/learn` captures it before the chat dies

## Related Forges

- **improve-forge** — finds gaps. learning-forge ensures the lessons persist.
- **brutal-forge** — finds bugs. learning-forge routes them to guardrails.
- **ship-forge** — checks readiness. learning-forge feeds gotchas back.
